## Description
A command-line Python application to track expenses. Users can:

- Add new expense entries (date, category, amount, and description)
- View all expenses
- Get total spending by category
- Delete an expense by entry number
- Exit the program

The data is stored in memory during runtime (not saved to a file or database).

## How to Run
1. Make sure Python 3 is installed.
2. Run the app using:

```bash
python expense_tracker.py
